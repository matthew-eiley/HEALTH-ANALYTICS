How to Read This Chart

Each square represents a day. The color intensity shows your performance level:

  • Dark green: Excellent performance (90%+ for sleep consistency, 92.5%+ for hours)
  • Medium green: Good performance (80-90% / 85-92.5%)
  • Light green: Fair performance (70-80% / 77.5-85%)
  • Very light green: Poor performance (60-70% / 70-77.5%)
  • Gray: Very poor performance or no data
